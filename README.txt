Instalação:

Certifique-se de ter Python instalado.
Salve os arquivos cliente.py e servidor.py no seu diretório de preferência.

Execução do servidor:

No terminal, navegue até o diretório onde o servidor foi salvo.
Execute o comando: python servidor.py.

Execução do cliente:

Em outro terminal, navegue até o diretório onde o cliente foi salvo.
Execute o comando: python cliente.py.
Escolha se deseja enviar um texto ou um arquivo.
